## 프로젝트 소개
>기존의 MPA(Many Page Application) SSR(Server Side Rendering) 방식은  
새로운 페이지로 이동할 때마다 페이지가 깜빡거리는 현상 발생한다.  
이를 보완하기 위해 CSR(Client Side Rendering) 방식을 활용하는
SPA(Single Page Application)를 적용한 쇼핑몰 사이트 개발
